# progtl
Programmers Tools
## Description
* ` fl          `- Hex dumper demo #1
* ` hxd.cpp     `- Hex dumper demo #2
* ` hxd.hpp     `- Hex dump the character arrays to stdout for debug... It can do strings or streams with pretty easy mods.
* ` Makefile    `- Special Makefile, it build all *.cpp in current directory.
* ` nlaeof      `- Trailing newlines at end of file adjusting in the source file. Work on the text file.
* ` wa.sh       `- Trailing whitespace at ends of lines remover, clean source files and look more Professional.
